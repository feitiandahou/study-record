# Database Study Record

This directory stores database-related study notes, executable demos, and interview preparation materials.

## Current Focus

- MySQL interview preparation
- SQL writing practice
- Performance analysis and troubleshooting cases

## Suggested Learning Order

1. SQL basics and execution order
2. Indexes and query plans
3. Transactions, isolation levels, and MVCC
4. Locks, deadlocks, and concurrency control
5. Logs and crash recovery
6. Slow SQL diagnosis and optimization
7. Replication, sharding, and common architecture topics

## Directory Layout

- `mysql/notes/`: topic-based notes
- `mysql/sql/`: schema, seed data, and practice SQL
- `mysql/interview/`: interview questions and scenario drills

## Output Standard

Each topic should try to include:

- a concise concept summary
- a few key interview questions
- one or more executable SQL examples
- common pitfalls or follow-up questions
