# MySQL Interview Prep

This section is organized for interview-oriented learning rather than full product development.

## Goal

Build a reusable knowledge base that covers both:

- concept explanation
- hands-on SQL practice

## Structure

- `notes/`: core concepts, organized by topic
- `sql/`: table definitions, sample data, and practice cases
- `interview/`: questions, answer frameworks, and scenario drills

## Review Strategy

1. Read one note.
2. Run the related SQL.
3. Summarize the answer in your own words.
4. Revisit with interview questions and scenario follow-ups.

## Priority Topics

- indexes
- transactions and MVCC
- locks and deadlocks
- explain and slow SQL optimization
- replication and sharding basics
