# Dijkstra demo

This module demonstrates a minimal Dijkstra shortest-path solution in Python:

- Work on graphs with non-negative edge weights.
- Compute the minimum total cost from a start node to a target node.
- Return both the total distance and the chosen path.

## Run

```powershell
uv sync
uv run python -m app.main
```

## Test

```powershell
uv run pytest
```
