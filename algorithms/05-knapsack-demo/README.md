# 0/1 knapsack demo

This module demonstrates a minimal dynamic programming solution for 0/1 knapsack in Python:

- Each item can be used at most once.
- Compute the maximum value within a capacity limit.
- Return only the best achievable value.

## Run

```powershell
uv sync
uv run python -m app.main
```

## Test

```powershell
uv run pytest
```
