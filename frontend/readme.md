# Frontend demos

This directory contains independent frontend study modules.

## Modules

- `01-react-task-board-demo`: practice React state updates, derived counts, and list rendering.
- `02-react-search-gallery-demo`: practice React effects, deferred filtering, and async UI states.

## With npm

Each demo is an independent Vite project.

```powershell
cd frontend/01-react-task-board-demo
npm install
npm run dev
```

Use the same pattern in any other module directory.
